## 🎨 Featured Project: ApexGym Management System
We have created a fully featured, client-side Gym Management Web Application (`ApexGym`) under `projects/gym-management-system`.

### How to Run ApexGym Locally:
1. **Open your terminal or command prompt (cmd)**.
2. **Navigate into the gym system project directory**:
   ```bash
   cd "projects/gym-management-system"
   ```
3. **Start the local server**:
   * **Using Python**:
     ```bash
     python -m http.server 8080
     ```
   * **Using Node (npx)**:
     ```bash
     npx http-server -p 8080
     ```
4. **Open your browser** and go to: `http://localhost:8080`

*Alternatively, you can simply open the [index.html](projects/gym-management-system/index.html) file directly in your browser by double-clicking it.*
